## About

This is my personal portfolio website — built to present my work as a data analyst and give potential clients or employers a clear picture of what I do and how I think.

The site features five projects built with Power BI, Excel, SQL, and Python, and is designed to be fast, interactive, and readable on any device.

## Tech Stack

This is a static site — no frameworks, no build tools, no dependencies.

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, animations, scroll-triggered effects
- **Vanilla JavaScript** — IntersectionObserver for reveals and counter animations, custom cursor
- **Google Fonts** — Syne, IBM Plex Mono, Lora
- **Hosted on** GitHub Pages

---

## Features

- Animated scroll reveals on all sections
- Skill bars that fill on scroll into view
- Stat counters that animate on load
- Custom cursor with trailing ring (desktop)
- Responsive layout for mobile and desktop
- Hover effects on project cards (image zoom + color reveal)
- Subtle grid overlay background
